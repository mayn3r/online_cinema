# Онлайн-кинотеатр

#### Установка
```bash
git clone https://github.com/mayn3r/online_cinema.git
cd online_cinema
```

#### Запуск через Docker
```bash
docker compose up -d --build
```

#### Запуск через `uv`
```bash
uv run main.py
```

###### Если `uv` не установлен, устанавливаем командой
```bash
pip install uv
```
