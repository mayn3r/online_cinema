О проекте

Учебный проект для производственной практики 2 курса

Что это за проект?

Онлайн-кинотеатр — это веб-приложение, разработанное в рамках производственной практики 2 курса по направлению «Кибербезопасность информационных систем» ДГТУ.

Проект демонстрирует полный цикл разработки современного веб-приложения: от проектирования базы данных и backend-логики до создания адаптивного пользовательского интерфейса. Основная цель — показать навыки работы с современными технологиями и фреймворками.

Приложение позволяет пользователям просматривать каталог фильмов, получать детальную информацию о каждом фильме, регистрироваться и входить в систему, а также управлять своим профилем. Для администраторов предусмотрена панель управления с возможностью управления пользователями и их балансом.

Цели и задачи

🎯

Проектирование БД

Разработка реляционной базы данных PostgreSQL с оптимальной структурой таблиц, связей и индексов

⚙️

Backend-разработка

Создание RESTful API на FastAPI с асинхронной обработкой запросов и JWT-аутентификацией

🎨

Frontend-разработка

Создание адаптивного пользовательского интерфейса с использованием HTML5, CSS3 и JavaScript

🐳

Контейнеризация

Настройка Docker и Docker Compose для удобного развёртывания и масштабирования приложения

🔒

Безопасность

Реализация безопасной аутентификации, хэширования паролей и защиты от распространённых уязвимостей

📊

ORM и миграции

Использование Tortoise ORM и Aerich для управления схемой базы данных и миграциями
